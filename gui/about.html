Cloud Masking

CloudMasking is a Qgis plugin for make the masking of clouds, cloud shadow, cirrus, aerosols, ice/snow and water for Landsat (4, 5, 7, 8, 9) products using different process such as Fmask, Blue Band, Cloud QA and Pixel QA.
